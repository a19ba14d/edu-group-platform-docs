# PRD（按需求文件拆分）

本目录为《集团化数字教育管理系统》的 PRD 文档体系：
- 以【Rxx-xxx】作为唯一追溯键；
- 每条需求对应一个独立文件：`req-Rxx-xxx.md`；
- 按平台 → 模块目录组织；
- 每条需求文件均包含：权限、`school_id` 数据范围、集团/学校视角差异（如适用）、审计要求、验收用例。

## 导航
- 通用规范：`PRD/common/README.md`
- 小程序（学生端）：`PRD/miniprogram-student/README.md`
- 小程序（家长端）：`PRD/miniprogram-parent/README.md`
- 小程序（教职工/教师端）：`PRD/miniprogram-teacher/README.md`
- Web 管理端（集团/学校双视角）：`PRD/web-admin/README.md`
- 基础预注册 H5：`PRD/h5-preregistration/README.md`
