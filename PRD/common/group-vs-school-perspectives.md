# 集团视角 vs 学校视角（双管理模式）

> 强制来源：`附件/01-三橙智星开发团队技术协议与开发规范.md`（`X-View-Scope` 规范）。

## 1. 视角定义
- **学校视角（School Mode）**：仅处理当前学校 `school_id` 的数据。
- **集团视角（Group Mode）**：集团人员可以：
  - `X-View-Scope: all` 查看全集团汇总（仅集团管理员/授权角色）
  - `X-View-Scope: {school_uuid}` 进入指定学校视角

## 2. 强制规则
- 必须校验：请求者是否有权访问 `X-View-Scope` 指向的学校。
- 集团人员进入学校视角的任何查看/修改必须写审计日志。
- 集团汇总场景必须避免泄露未授权学校的明细数据。

## 3. 与单条需求文件的关系
- Web 管理端的每条需求必须写明：该功能在 `all` 与 `school_uuid` 两种 scope 下的可见/可操作差异。
