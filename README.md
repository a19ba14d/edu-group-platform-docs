1. 打开 [PRD/traceability-matrix.md](PRD/traceability-matrix.md)
2. 按【Rxx-xxx】搜索并点击对应链接
3. 在目标 PRD 文件中查看该条目的权限、school_id、视角差异、审计与验收用例
